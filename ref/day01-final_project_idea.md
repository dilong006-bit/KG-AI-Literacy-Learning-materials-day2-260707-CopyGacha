# PRD - 카피라이팅 헤드라인 생성기

## 1. 프로젝트 개요
- 제품명: HeadlineRoulette (가칭)
- 한 줄 설명: 제품 키워드를 입력하면 여러 카피 공식에 맞춰
  마케팅 헤드라인을 자동 생성해주는 웹앱
- 목적: 백지 상태에서 카피를 짜는 대신, 공식별 시안을
  한 번에 받아 빠르게 다듬기
- 타깃 사용자: 마케터, 기획자, 1인 사업자
- 실행 환경: 브라우저 (HTML 단일 파일, 서버/DB 없음)

## 2. 핵심 기능
1. 키워드 입력
   - 제품/서비스 키워드, (선택) 타깃 고객, (선택) 핵심 혜택 입력
2. 공식별 헤드라인 생성
   - 숫자형 / 질문형 / 혜택형 / 공포소구형 / 호기심형 등
     5~6개 공식으로 각각 헤드라인 생성
   - "다시 생성" 시 같은 공식으로 다른 표현 출력
3. 결과 관리
   - 마음에 드는 헤드라인 즐겨찾기(별표)
   - 클릭 시 클립보드로 복사
4. (선택) 룰렛 연출
   - 생성 버튼 클릭 시 짧은 애니메이션 후 결과 표시

## 3. 화면 구성
- 상단: 제목 + 입력 폼(키워드 / 타깃 / 혜택 / 생성 버튼)
- 중앙: 공식별 결과 카드 목록 (공식명 배지 + 헤드라인 + 복사·별표 버튼)
- 하단: 즐겨찾기한 헤드라인 모음 영역

## 4. 기술 요구사항
- HTML + CSS + Vanilla JavaScript (프레임워크 없음)
- 파일 구조: index.html / style.css / script.js 3개로 분리
- 외부 라이브러리·API 호출 없음 (전부 클라이언트에서 규칙 기반 생성)
- 반응형: 모바일에서도 카드가 깨지지 않게

## 5. 파일 구조
- index.html   (화면 마크업)
- style.css    (스타일)
- script.js    (헤드라인 생성 로직 + 이벤트 처리)

## 6. 예외 케이스 (Validation)
- 키워드가 비어 있을 때: 생성 막고 안내 메시지
- 키워드가 너무 길 때(예: 50자 초과): 잘라내거나 경고
- 특수문자만 입력했을 때 처리
- 같은 결과가 반복 생성되지 않도록 표현 다양화
